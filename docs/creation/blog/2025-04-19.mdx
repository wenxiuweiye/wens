## 📖 知识

### 1. markdown table 换行

今天在用 rspress 写 markdown 表格的时候，发现不是所有 markdown 渲染器 都支持解析 table 时夹杂 HTML 标签。

rspress 解析 md/mdx 时是使用 @rspress/mdx 插件，而 @rspress/mdx 就是 @mdx-js/mdx 的 rust 重构的版本。那么可能类 @mdx-js/mdx 的库都不支持解析 table 时夹杂 HTML 标签。

通用的解决方案：

**用空白的单元格来填充示意它和之前的单元格是同一个值**

```md
| 姓名 | 简介                       |
| ---- | -------------------------- |
| 张三 | 这是张三的简介。           |
|      | 他是一个优秀的软件工程师。 |
| 李四 | 这是李四的简介。           |
|      | 他热爱开源技术。           |
```

### 2. rspress auto navbar

我制作应用时用了 trae ，他在生成 rspress 的导航栏时，用的 rspress.config.ts，结果导致项目无法自动生成侧边导航栏。后面根据官网的说法：

**_在 rspress.config 里配置了导航栏会阻止 rspress 根据 \_meta.json 自动生成侧边导航栏_**
